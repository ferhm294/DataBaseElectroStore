# DataBaseElectroStore

## Descripción
Este proyecto consiste en la creación de una base de datos relacional para una tienda de electrodomésticos. La base de datos incluye diversas entidades relacionadas con los productos, las marcas de los productos, los clientes, las tiendas, los empleados, los cargos de los empleados, las facturas, los ítems de las facturas y una tabla de inventario.

## Tecnologías Utilizadas
- **Base de datos**: MySQL para DDL y SQL Server para DML
- **Entorno de desarrollo**: MySQL Workbench y SSMS

## Ramas
- **DDL**: contiene el esquema en MySQL.
- **DML**: contiene la inserción de datos y consultas. Para esta rama se usaron insumos proporcionados por la UNED por lo que no hay un esquema en SQL Server. El modelo relacional de la base de datos es igual al de la rama DDL pero en SQL Server.
